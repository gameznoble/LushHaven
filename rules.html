Server Rules

Enforced across all LushHaven Network platforms.

Chat & Community

Character Spam
Sending long strings of repetitive characters or symbols (e.g., "aaaaaa").
Mute (Tier 1 Ladder)
Similar/Unique Spam
Sending 3+ similar messages or flooding chat with different messages quickly.
Mute (Tier 1 Ladder)
Command Spam
Abusing chat-triggering commands like /afk, /bounty, or toggle alerts.
Mute (Tier 1 Ladder)
Foreign Languages
Using non-English languages in public chat. Please use private messages.
Warning / 6h Mute
Impersonation
Claiming to be a staff member, developer, or another player.
1d Mute / Ban

Using disallowed modifications (Killaura, Fly, X-Ray) to gain an advantage.
Permanent Ban

Destroying builds or stealing items within 20 blocks of a claim.

Alt Account Limit
Bypassing the 5-account limit per player/household via VPN or Proxy.
30d Ban / Permanent
Scamming
Misleading players in trade deals or Auction House (renamed items).
Permanent Ban

Zero Tolerance

Doxing / DDOS

Discrimination
Using slurs or marginalizing players based on identity, race, or sex.
Permanent Ban
